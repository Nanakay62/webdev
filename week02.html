Programming Basics

Programming is about making a computer do what you want. Making a program means giving a computer a set of instructions to perform a task.

Comments

A piece of code ignored by the language. Comments are made to make anyone who is reading the code understand what is going on. In Javascript, there are two types of comments. They are;

  • // for single line comments
  • /* & */ for multi-line comments
Writing comments in code is good practice.

JS Grammer

Javascript is made up of statements that end with a semicolon(;) or with a new line. It is not mandatory to use a semi-colon to statements written in JS, since the language has an Automatic Semi-colon Insertion Process (ASI). This process attempts to place semi-colons at the end of each statement. This however, may lead to errors and cause automated services like minifiers and validators not to work properly.

Block

A block is a series of statements collected inside curly brackets.
Example;
{
const = "Hello world";
alert(message);
}

Reserved Words

Reserved words are ones that cannot be used as variable names. These include; char, default, byte, boolean, delete etc.

Variables

They are used to refer to a value that is stored in memory Javascript uses "const" and "let" to declare variables. The "=" sign is used to assign a variable When a const is used, a variable cannot be reassigned, however, when "let" is used, the variable may be reassigned. Both are block scoped and hence, exist inside the blocks in which they are declared.
A variable that starts with an underscore _ refers to private properties and methods. $ is used in JQuery.

Soft Equality and Hard Equality

Soft Equality has double Equal signs == whilst Hard Equality has triple equal signs ===.

ARRAYS

An array is an ordered list of values that are created by using a pair of square brackets. A constructor function, new Array() can be used, how ever, a pair of square brackets [] are preferred because it involves less typing.
The length property is used ti determine the length of an Array. The join() method can turn the array into a string that compresses all items separated by commas.

Sets

A set is a data structure that contains a unique set of values that cannot contain duplicates. The number of values in a set can be determined using the size method.

Functions

A function is code that can be referenced by name and is almost like a small self-contained mini-program.

Defining Functions

Functions can be defined in three main ways. They are; functional declarations, expressions and arrow methods.

Invoking A function

Invoking a function means to run a code inside the function's body. invoking is done by typing the name of the function, and followed by a parentheses. When using functions, there is no need to write repetitive blocks of code and all functionality is kept at one place. An example is; hey(). Calling the function name followed by the parentheses is invoking.

IReturn Values

Return statements are used to return the value of functions. A function that does not return anything returns "undefined" as default.

Parameters and Arguments

They represent values provided for a function as an input. The difference between them are that, a parameter is the variable listed inside the parentheses in the function definition and an argument is the value that is sent to the function when it is called.
